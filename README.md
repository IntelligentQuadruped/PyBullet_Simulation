# PyBullet_Simulation

### Control Parameters

Input parameters:

```
params = [  Scalling factor outside motors [rad],
			Scalling factor for inside motors [rad],
			Phase difference between legs [rad],
			Offset factor for leg 1 to leg 0,
			Offset factor for leg 2 to leg 0,
			Offset factor for leg 3 to leg 0
			]
```
